This post is about a pretty small feature of CSS. We've talked about transformations in the past, and this is building on that. A new transformation I found was scaleX(). This function resizes an element on the x-axis. So, with the ball seen above, when it reaches the lowest point, it stretches out appearing more realistically weighted than if it didn't stretch out.

  .bounceball:before {
    position: relative;
    content: '';
    display: block;
    top: 0;
    width: var(--width);
    height: var(--height);
    border-radius: 50%;
    background-color: #fbae17;
    transform-origin: 50%;
    animation: bounce 500ms alternate infinite ease;
  }
  
  @keyframes bounce {
    0% {
      top: var(--bounce_height);
      height: 5px;
      border-radius: 60px 60px 20px 20px;
      transform: scaleX(2);
    }
    35% {
      height: var(--height);
      border-radius: 50%;
      transform: scaleX(1);
    }
    100% {
      top: 0;
    }
  }
