For this week, I followed the tutorial found Here on how to create both a 3D object AND how to animate it. This makes use of @keyframes and "animation" which were used in a previous blog with the addition of "transform", "transform-style", "perspective", and "perspective-origin". 

        .front {
          transform: translateZ(1em);
        }
        
        .back {
          transform: translateZ(-1em);
        }
        
        .left {
          transform: rotateY(90deg) translateZ(-1em);
        }
        
        .right {
          transform: rotateY(90deg) translateZ(1em);
        }
        
        .top {
          transform: rotateX(90deg) translateZ(1em);
        }
        
        .bottom {
          transform: rotateX(90deg) translateZ(-1em);
        }

Before the selectors above, 6 div's are created inside a container "cube". Now, each of those attributes are transformed so that they simulate a cube. If only these transormations happen, it will not look right because by default, "transform-style" is set to "flat". Therefore, any 3D motion is not noticed. By setting it to "preserve-3d", we see the changed we made correctly.
