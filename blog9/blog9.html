For this blog, I'll be focuing on getting a better understanding of faking pages with React. Specifically, I'll be focusing on making buttons link to "other pages". So, this will be using the button's OnClick() function and react-router-dom's navigate().

First, in App.js, you need to use react-router-dom's Routes and Route. a Route essentially is a linkable Component that goes to a pseudo-path. So, instead of needing a path to an html file, this path leads to a specific Component.

    <Routes>
      <Route path="" element={<Home />} />
      <Route path="/page1" element={<Page1 />} />
      <Route path="/page2" element={<Page2 />} />
    </Routes>

A fun thing I found with this you can see is to create a home page, the path is "". It's pretty obvious but I thought it was fun. Next in the Home component, I have two buttons that, when clicked, take you to Page1 or Page2 respectively.

    <button onClick={()=> { navigate(`/page1/`); }}>
      Page 1
    </button>
    
    <button onClick={()=> { navigate(`/page2/`); }}>
      Page 2
    </button>

navigate(), working with Route, essentially create links to another "page". When navigate() changes the path, the Route defined with that path then jumps to the content in element specified (Page1 or Page2).
